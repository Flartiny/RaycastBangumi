# RaycastBangumi

RaycastBangumi 是一款 [Raycast](https://raycast.com) 扩展，让你无需打开浏览器即可快速检索、浏览和管理你的 [Bangumi](https://bgm.tv) 数据。

## 功能

### 🔍 Bangumi Search
搜索 Bangumi 上的条目（动画、书籍、音乐、游戏、三次元），支持按类型筛选。点击条目可查看详细信息和制作人员、声优阵容。

### 📅 Bangumi Calendar
查看本周的放送日历，按天浏览每日的播出条目。不需要登录即可使用。

### 📋 Bangumi List
浏览你的收藏（想看、在看、看过、搁置、抛弃），支持分页翻页。需要登录 Bangumi 账号。

### 进度管理与收藏
在条目详情页中，你可以：
- 调整观看进度（支持快捷键）
- 将条目标记为想看/在看/看过/搁置/抛弃
- 标记为"看过"时可选同时更新全部进度

## 使用

安装后，通过 Raycast 输入以下命令：

| 命令 | 说明 |
|------|------|
| `Bangumi Search` | 搜索条目 |
| `Bangumi Calendar` | 本周放送表 |
| `Bangumi List` | 我的收藏 |

使用前需要先登录。在未登录状态下运行会提示登录，按提示完成 OAuth 授权即可。

## 设置

在 Raycast 扩展配置中可以调整以下选项：

- **确认切换为「在看」** — 调整进度时，如果当前条目状态不是「在看」，询问是否切换（默认开启）。未收藏的条目始终会询问。
- **进度追满自动标记「看过」** — 当进度达到总集数时自动将条目标记为「看过」，无需确认（默认关闭）。
- **标记「看过」时确认更新进度** — 手动标记为「看过」时，询问是否同时将进度更新为总集数（默认开启）。
